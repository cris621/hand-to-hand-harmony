Introducción

En la actualidad, diversas plataformas y aplicaciones, especialmente los videojuegos se han creado para proporcionar a sus usuarios una experiencia de calidad al momento de ejecutarlos desde la comodidad de su móvil u ordenador, no obstante, los juegos más populares en la red no pueden ser disfrutados por ciertos grupos debido a sus diversidades funcionales.

LEER MAS

Entre estos grupos se encuentra la comunidad sordaue en general no acceden a varias actividades o no pueden realizarlas como se prevé en un inicio.

Considerando esta realidad, nos hemos propuesto a desarrollar un videojuego que aborda la forma básica de comunicación de la Comunidad Sorda, de tal manera que se pueda interactuar en el juego con el abecedario en señas, sustituyendo a los controles comunes del teclado que se usan en juegos de PC.

El videojuego será destinado principalmente para los integrantes del Grupo de Estudio de Aproximación a la Sordera, no obstante, también puede ser manejado por familias que quieran y busquen aprender a relacionarse con una persona no oyente, utilizando el método de mando propio del juego.

Debido a que queremos contribuir en normalizar la integración de esta comunidad en la sociedad, nuestro videojuego no tendrá fines de lucro, puesto que lo vemos como una herramienta capaz de aportar a algo más grande y beneficioso para todos.

“Una persona sorda puede hacer cualquier cosa igual que un oyente, excepto oír” (King Jordan)

Resumen

Hand-to-hand Harmony, es un videojuego 2D, orientado y conceptualizado en la Comunidad Sorda. Su creación pretende ser una alternativa para aquellos que deseen aprender sobre el Alfabeto Dactilológico de la Lengua de Señas Ecuatoriana, de una manera sencilla e interesante, para así crear una cercanía con conocidos y familiares que formen parte de esta comunidad.

LEER MAS

El presente producto se compone de dos programas: un teclado virtual y el videojuego. Por una parte, el teclado virtual fue desarrollado en Phyton, lenguaje de programación que permite el uso de librerías necesarias para la detección de gestos dactilológicos, por medio de una serie de coordenadas, líneas y puntos; dejando de lado el manejo del teclado, e implementando el uso de cámara. Mientras que el videojuego, utiliza este medio para controlar el movimiento de los personajes, en base al alfabeto dactilológico de la Lengua de Señas Ecuatoriana, exceptuando las letras que requieran un movimiento en particular.

Hand-to-Hand Harmony se desarrolla en una historia contada a través de varias pistas y pinturas (minijuegos) que el protagonista, un joven de una pequeña aldea, empieza a encontrar tras recibir una misteriosa carta la cual le informa que es el elegido para descubrir y divulgar la verdad acerca de la creación de los humanos.

Sinopsis

La historia comienza con el protagonista despertando y encontrando una carta sobre su mesa. En la carta, se revela la verdad acerca del origen de la humanidad y se le informa que él es el elegido para compartir la leyenda de Hand-to-Hand Harmony con el mundo.

LEER MAS

A medida que se adentra en una emocionante aventura, deberá descubrir cada aspecto de esta historia y evitar que el malvado culto, que ha mantenido en secreto esta verdad durante tanto tiempo, lo detenga.

Guía de uso y mantenimiento

1. Instalación de las librerías y dependencias necesarias:

En primera instancia, se debe de tener todas las librerías y dependencias necesarias para ejecutar el programa. En el código proporcionado se utilizaron las librerías `cv2`, `numpy`, `pyautogui`, `pickle`, `sklearn`, `imutils` y `time`. Es importante verificar que todas estas librerías estén instaladas en el equipo antes de continuar.

LEER MAS

2. Entrenamiento del modelo

Antes de ejecutar el programa, se debe entrenar un modelo para reconocer las señas que se utilizarán en el juego. En el código proporcionado, se utilizó un modelo SVM pre-entrenado con el dataset de señas de la Universidad de Washington. En caso de que se desee utilizar señas personalizadas, se deberá entrenar un modelo propio. Para ello, se pueden utilizar herramientas como TensorFlow o Keras para entrenar un modelo de aprendizaje profundo o utilizar otras técnicas de aprendizaje automático para entrenar un modelo SVM o similar.

3. Configuración del programa:

Una vez que se tiene un modelo entrenado, se procede a configurar el programa para que pueda reconocer las señas específicas que se utilizarán en el juego. En el código proporcionado, se definen las señas a utilizar y se configura el modelo para que las reconozca. Se puede modificar este código para agregar o quitar señas y ajustar los parámetros del modelo según sea necesario.

4. Calibración de la cámara:

Antes de empezar a utilizar el programa, se requiere calibrar la cámara que se utilizará para capturar las señas. En el código, se usa una función de calibración para ajustar la cámara y asegurarse de que las señas se reconozcan correctamente.

5. Ejecución del programa:

Una vez que se ha realizado la calibración y configuración necesarias, se puede ejecutar el programa para empezar a reconocer las señas y utilizarlas como entrada de teclado virtual en el juego. Las señas reconocidas se interpretan como entrada de teclado virtual utilizando la librería PyAutoGUI.

6. Manejo en el juego:

Para usar el teclado virtual dentro del videojuego, se deberá asegurar de que el juego esté configurado correctamente para recibir la entrada del mismo. Dentro de este último, la función `pyautogui.press` simula las pulsaciones de teclado en el juego, que se configuran según los controles del juego los cuales son: “A”: izquierda; “E”: derecha; “V”: subir; “W”: bajar; “L”: interactuar y las que se mencionaron en el último apartado del proceso de elaboración del teclado virtual que se utilizan para escribir dentro del videojuego.

7. Descarga y uso del producto:

En cuanto al uso del videojuego, una vez ingresado a la página web, se deberá ir al apartado de descarga e instalar el teclado virtual y el videojuego, los cuales requieren un sistema operativo Windows de 64 bits. Al ser ejecutables, si es un usuario que quiere utilizar el producto sin necesidad de realizar cambios dentro del mismo, no es indispensable la instalación de ningún otro programa, solo el uso de una cámara.
